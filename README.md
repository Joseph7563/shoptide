# 🛒 Shoptide

Shoptide is a modern, user-friendly eCommerce website designed to make online shopping simple, fast, and reliable. Browse a wide range of products, discover the latest deals, and order directly via WhatsApp for instant communication and seamless transactions.

## 🌍 Live Website
https://joseph7563.github.io/shoptide/

## ✨ Features
- Clean and responsive product layout
- Mobile-friendly design
- Direct WhatsApp ordering system
- Fast and lightweight (no backend required)
- Easy to customize and expand

## 🧰 Technologies Used
- HTML5
- CSS3
- JavaScript

## 📞 Contact
For orders and inquiries, customers can reach us via WhatsApp directly from the website.
